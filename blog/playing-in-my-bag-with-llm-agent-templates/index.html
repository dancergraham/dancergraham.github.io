Playing "In my bag..." with LLM agent templates

written by Graham Knapp on

There's a memory game "In my bag..." where you pretend you have a list of things in your bag. The next person in the circle has to list all the same things and add one more at the end. The game ends when someone fails to remember the previous list.

It occurred to me that this is a great analogy for the templates used in LLM chatbots! Every time you chat with an LLM the LLM re-reads the whole history of the conversation before responding. This is why they get slower over time, particularly if they are making web searches, viewing images, using MCPs or other tools which add a lot of hidden tokens to the script.

The ollama docs describe a simple template here ... https://github.com/ollama/ollama/blob/main/docs/template.md
